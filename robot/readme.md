        Gazebo相比前两者还是差了些.
        OpenHRP主要是做humanoid仿真的.
        Gazebo在学习ROS的时候可能会用到，比如跑一跑Turtlebo的仿真，对于学习还是很有帮助的。
        VREP对用户友好很多，文档齐全，EDU版本也没有功能限制，还是跨平台的，所以初学者容易上手很多。
